About

Untitled Unity Game is a genre-defining cover for a game I'm working on that I don't want my real name associated with anywhere on the publicly accessible internet so you get this instead. As Player challenge yourself against Enemies in a never before seen adversarial experience where you input keyboard controls to do actions. There are only three things you need to know to win and to fulfill the <ol> requirement:

  1. Press the right buttons
  2. Don't press the wrong buttons
  3. Win

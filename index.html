About Me

Hello, I am a computer science student at UCSD with an interest in software engineering, particular in the fields of web and game development. I love videogames, which provided my first explorations into programming through modding my favorite games and developing one of my own in Unity. Beyond gaming, I've expanded my interest into web development, initially as a way of promoting my game but now with the ambition to create quality websites that put the user first.

I've had exposure to both backend and frontend development, with a preferred tech stack of C# using ASP.NET Core and Vue.js with PostgreSQL. Much of my C# knowledge comes from experience developing my game in Unity engine, but I've had further practice in unrelated personal projects and in backend and API web development. C++ is another favored language of mine for applications outside of web and game development. I concede that I'm not an expert in the technologies I practice, so I'm always seeking to learn more to improve my work and create greater things.

Be sure to check out some of my (always expanding) works in Projects or Games for a better look into what I can do or if you just want a game to play. I'm always eager to discuss personal projects of mine with those interested; if that's you, don't hesistate to Contact me.
